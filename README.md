
## 基本记录

# 整体框架
- app           管理平台页面集合
  - views/layout 布局属于views下面所有页面的共有布局
- components    组件集合
- ui            所有app中用到的组件，目的是为了减少自动生成的页面
- layout        布局属于views下布局的样式与组件
- libs          公共的处理方法集合
- styles        样式文件夹，公共的样式文件存放
- theme         样式主题 主要是针对antd的
